## 核心规程 (Core Rule)
本工作流运行 `Skill Audit` 阶段。目标是:检查当前技能体系、规则体系、工作流体系是否存在能力缺口,并决定是否引入、替换或更新。

## 共享知识库公约 (Shared Knowledge Location)
- 7 个工作流共用一套单层跨项目共享知识库:`C:\knowledge`。
- `C:\knowledge` 只保存跨项目长期知识(协作规则、编码偏好、设计偏好、长期经验、流程偏好)。
- 项目结构、模块边界、接口契约、业务规则、项目技术决策必须维护在当前项目目录内(如 `docs/architecture/`),不写入 `C:\knowledge`。
- 禁止在 `C:\knowledge` 中创建 `global/` / `project/` / `global_workflows/` 目录或 core/stages 等冗余副本。

## 必读规程 (Required Reading Procedure — 固定 7 步)
> 本节定义本入口必须触发的物理读取动作。门禁细则与禁止行为见 `core/stage-gates.md` 与 `stages/7-skill-audit.md`,本入口不重抄。

1. **加载规则**:物理读取本入口列出的"必须加载"中所有 `core/` / `references/` / `stages/` / `templates/` 文件,登记到 Evidence Ledger。
2. **回读上游产物**:物理读取最近的 `.workflow/retro-report.md`(若存在),登记 `E-id`;若用户主动触发巡检且无 retro,标注 `Not Applicable; user-initiated`。
3. **验收输入**:在产物 `Upstream Consumption` 章节逐条标注最近 Retro Report 的所有 `F-id` 与 `Change-id`(`Consumed / Not Applicable / Deferred`)。
4. **探索源码、图谱、知识**:按 `stages/7-skill-audit.md` 中查询层默认 `Skipped` 规则处理 codegraph(初始化层不豁免);物理读取 `C:\knowledge\INDEX.md` 与候选技能 `SKILL.md`(若有),登记 `E-id`;若涉及候选技能与现有代码符号兼容性核对,可调用 `codegraph.search_symbol`。
5. **产出文件**:按 `templates/skill-audit-report.md` 创建或更新 `.workflow/skill-audit-report.md`,Gaps Found / Candidate Skills / Decisions 必须分配 `Gap-id` / `Cand-id` / `Decision-id`,4 维评估(Benefit / Compatibility / Maintenance Cost / Workflow Disturbance)逐项填写。
6. **回读自检**:执行 `core/anti-hallucination.md` 中的 Verification Order;核对每个 `Decision-id` 必须挂高置信 `E-id`;不能区分技能/流程/知识缺口时不得下结论。
7. **下一步建议**:若决策为 Install / Replace / Update 且无未决问题则闭合并执行决策;若问题本质是流程缺陷则 `/6-复盘优化`;否则只写 `Blockers` + `Rollback Advice`。

## 阻断违规与门禁细则
本入口的硬门禁、禁止行为、伪证据判定与回退条件统一由以下文件承载,**禁止在本入口重抄**:
- `core/stage-gates.md`:7 步执行序列、Input / Scope / Evidence / Upstream Consumption / CodeGraph / Exit / Rollback Gate。
- `core/anti-hallucination.md`:伪证据黑名单、Claim Evidence Map、工具失败语义、Sequential Thinking 回包字段要求。
- `core/codegraph-engine.md`:初始化层强制规则、本阶段查询层可结构化跳过条款。
- `core/evidence-engine.md`:证据台账 6 列结构、高置信结论规则。
- `stages/7-skill-audit.md`:本阶段 Allowed / Disallowed Actions、Required Output、Handoff Checks、Rollback Conditions。
- `references/tool-compatibility.md`:候选技能兼容性评估口径。
