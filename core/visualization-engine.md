# Visualization Engine

## Purpose
为前端相关任务提供结构级可视化能力，用于表达布局、信息层级、交互路径和状态变化，而不是做高保真视觉设计。

## Trigger Conditions
满足以下任一情况时，应考虑生成结构原型页：
- 新页面
- 页面整体改版
- 多区块布局
- 工作台、仪表盘、复杂表单
- 组件层级复杂且纯文字难以说明
- 用户明确表示文字描述不直观

## Scope of Visualization
结构原型只允许表达：
- 布局结构
- 信息结构
- 交互路径
- 状态变化

## Explicit Exclusions
以下内容不属于结构原型范围：
- 高保真视觉设计
- 品牌化样式系统
- 装饰性细节优化
- 与结构说明无关的复杂动画

## Output Forms
可接受的产物形式：
- 结构说明文档
- 可访问的简单 HTML 原型页
- 极简前端原型页

## Stage Integration
- 方案设计阶段：用来降低结构理解歧义。
- 任务拆分阶段：用来辅助拆分页面结构任务。
- 审查阶段：用来检查实现结构是否偏离原型。

## Result Check
原型页是否有效，不看美观度，只看它是否让结构、交互和状态表达更清晰。
